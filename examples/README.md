# Examples

These examples make the MCP protocol visible.

- [Raw JSON-RPC transcript](rpc-transcripts/basic-flow.md) shows the messages for
  initialization, tool listing, tool calling, resource reading, and prompt use.
- [Minimal stdio Python server](minimal-stdio-python/README.md) shows how JSON-RPC
  messages can move over stdin and stdout.

The Python server is intentionally small and educational. Use an MCP SDK for real
servers.
